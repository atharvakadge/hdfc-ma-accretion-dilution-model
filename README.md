# HDFC Bank – HDFC Ltd. M&A Accretion/Dilution Model

## Overview
An M&A accretion/dilution model analyzing the ₹40B all-stock merger of 
HDFC Bank and HDFC Ltd. (July 2023) — India's largest ever corporate merger.

## Key Question
Does the combined entity's EPS rise (accretive) or fall (dilutive) vs 
HDFC Bank's standalone EPS?

## Deal Structure
- Type: Reverse merger (subsidiary absorbs parent), 100% stock
- Swap Ratio: 42 HDFC Bank shares per 25 HDFC Ltd shares (1.68x)
- New Shares Issued: ~311 crore

## Key Findings
| Metric | Value |
|--------|-------|
| HDFC Bank P/E (consolidated) | 19.5x |
| HDFC Ltd P/E (consolidated) | 20.2x |
| Standalone EPS | ₹82 |
| Combined Pro-Forma EPS | ₹83 |
| Accretion/(Dilution) | +0.8% (EPS-neutral) |

## Key Insight
With near-identical P/E ratios, the deal was mathematically EPS-neutral. 
This confirms the merger was driven by strategic and regulatory rationale 
(cross-sell, cheaper funding, NBFC regulatory pressure) rather than 
immediate EPS accretion.

## Concepts Demonstrated
- Accretion/dilution mechanics for all-stock deals
- The P/E rule: Acquirer P/E vs Target P/E determines outcome
- Standalone vs Consolidated figures (and why mixing them distorts P/E)
- Synergy sensitivity analysis

## Tools
Microsoft Excel — cross-sheet linked model with deal-driven share issuance
